Project title: TASTYSEARCHPrerequisite: XAMPPInstructions on launching project:
* install latest XAMPP software 
* open XAMPP, turn on Apache server and MySQL on action part
* then download our project file (TastySearch) and unzip it to 'htdocs' folder on XAMPP folder 
* go to localhost/phpmyadmin on your browser and create project named tastysearch then import tastysearch_sql.sql file to database (which is also in project folder)
* then open your browser and write following: localhost/tastysearch/tastysearch.php
* then you can see 4 canteens in dropdown menu, so now you can navigate through website
